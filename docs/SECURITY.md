# Безопасность

## Репозиторий (публичная версия)

- Админ-панель в репозиторий **не входит** (см. `.gitignore`). На GitHub публикуется только портал для учеников (классы, предметы, параграфы, тесты). Админка используется только локально.

## Портал (index.html)

- Вывод контента из JSON и из localStorage экранируется через `escapeHtml` (защита от XSS).
- Обработчики вешаются через `addEventListener`, в разметку не подставляется код из данных.
- Сообщения и подсказки выведены в контейнер с `aria-live="polite"` для скринридеров.
